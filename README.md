# Aplikasi DapoerJuna

DapoerJuna adalah chatbot interaktif berbasis LLM (Gemini) yang membantu kamu menemukan resep masakan Indonesia berdasarkan mood, bahan yang tersedia, diet, tingkat kesulitan, dan lain-lain. Chat bot ini juga dapat menyarankan resep paling disukai dan memberikan estimasi waktu memasak. Aplikasi ini juga dilengkapi dengan dashboard analisis interaktif serta informasi pendukung bagi UMKM kuliner untuk membantu mengembangkan bisnisnya.
